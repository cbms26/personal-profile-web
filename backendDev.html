Backend Development Concept

What is back-end?

Backend development mainly consists of:

  1. Server: is like a computer, that is usually runs 24/7 unless it's run on a local server.
  2. Application: logic applied on a computer/server.
  3. Database: to store data.

It is important to understand that Web Page differs from Web Application.

Some commonly used backend tools & its framework

  • Java - Spring
  • Ruby - Rails
  • PHP - Laravel/Cake
  • C# - ASP.net
  • Python - Flask/Django
  • JavaScript - Node.js
